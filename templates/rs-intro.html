Introducing Remote Services Container

Mycroft Remote Services is a docker container consisting of Mycroft Core, Apache and Remote Services Dashboard, It aims to provide users with a turnkey solution for getting Mycroft running on your remote device instantly.

This docker container runs Mycroft Core as a service inside its own environment and currently only provides a method to talk to the Core and GUI messagebus and Mycroft Skills interface for remote applications like Mycroft-GUI, This container does not support remote STT, TTS features. The container is highly aimed at devices running Mycroft GUI application on Android which provides its own inbuilt speech-to-text and text-to-speech services.

You Have Been Warned:

Mycroft Core Remote Services have no built-in security measures by default. We strongly recommend that you do not expose the services and ports outside a secure LAN/PAN unless you have other security measures in place.
